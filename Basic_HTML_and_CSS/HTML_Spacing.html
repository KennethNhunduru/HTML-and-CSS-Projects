There are 3 languages we need to learn to be able to create functional web pages and these are:

  1. HTML to define the content of the web page
  2. CSS to specify the layout of the page
  3. JavaScript to program the behaviour of the web pages

    4. HTML Language

      HTML, an acronym for HyperText Markup Language, is a primary markup language for creating websites.
    It consists of a series of codes used to structure texts, images, and other content to be displayed in a browser.
    Elements are the fundamentals of HTML. Each HTML document is made of elements, which are defined using tags.
    Tags are used to structure website content (text, hyperlinks, images, media, etc).
    Tags only "instruct" browsers how to show the content of the web page.

    CSS Language

      CSS stands for Cascading Style Sheets.
    It is a language designed to format the layout of Web pages.
    It is used to enable the separation of presentation and content, which includes layout, colors, and fonts.
    CSS enables the specific look of HTML in the browser.
    CSS is a great time-saver; you can write CSS once and then use the same sheet in multiple HTML pages.

    JavaScript Language

     JavaScript is a scripting or a programming language, allowing developers to perform complex features on web pages.
    JavaScript programs the behaviour of web pages.
    In JavaScript, the programs are called scripts.
    Basically, it can run on any device that has a specific program known as the JavaScript engine.
